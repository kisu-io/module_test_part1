4. CSS

Q26. Explain box model elements.

Box model is a box that wrap around any HTML elements and can be style with CSS. The box model have 4 elements: margin, padding, border, and the content itself.

Content - its what include in the box (texts, images, etc...).

Padding - defines the area around the content. Padding is transparent.

Border - A border that wrap around the content and padding.

Margin - defines the area outside of the border. Margin is trasparent.

Q28. What is vw/vh?

vw - is a unit that is defined in CSS and relative to 1% the width of the viewport.

vh - is a unit that is defined in CSS and relative to 1% the height of the viewport.

Q29. List all values of position property? Which one is default value?

This list of values of position property is:

static - this is the default value of position property, this value mean that the element being specified is positioned according to the normal flow of the document. It has no affected positioning with top, right, bottom, left and z-index.

relative - is same with static but in this value, all the affected positioning value will work such as: top, right, bottom, left and z-index.

fixed - this value will remove the element from the flow of the document, also fixed elements will always be relative to the document and finally it is not affected by any scrolling

absolute - this value will remove the element from the flow of the document, with all the positioning porperties affected it but all the other elements ignoring it.

sticky - this value will behave like relative value untill user scoll to a pre-defined location of the viewport. At which point, the element will be fixed to an area.

inherit - this value will not "cascade", it is used to make the child element follow the positioning of its parent.

Q31. List all values of justify-content property? Which one is default value?

The list of values of flex-direction property is:

flex-start - this is the default value of justify-content items are aligned towards the start line.

flex-end - tems are aligned towards the end line.

center - items are centered along the line.

space-between - items are ordered evenly in the line, first item at the start of the line and last item at the end of the line.

space-around - items are ordered with equal space between them but not including the start line and end line space. This means that there will be different spacing between the item and the edge.

space-evenly - items are ordered with equal space around them and including the start and end space.
